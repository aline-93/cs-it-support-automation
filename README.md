# Customer Support & IT Automation Workflow

This repository showcases a real-world automation workflow I designed to improve visibility, tracking, and organization of Customer Support and IT-related activities.

The solution was built using Microsoft Power Automate and Microsoft Forms, focusing on operational efficiency rather than code-heavy development.

## ðŸ”„ Workflow Summary

**High-level flow:**

1. Support interaction occurs (e.g. customer contact)
2. Internal Form is filled with clear checkboxes and fields
3. Power Automate processes the submission
4. A task is automatically created in Microsoft Planner
5. Activity is logged and can be reviewed by management
